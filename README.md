# Pesquisa de Opinião - Atendimento ao Cliente

## 📌 Descrição
Este projeto foi desenvolvido em Python para coletar a opinião de clientes sobre o atendimento de uma empresa.

## ⚙️ Funcionalidades
- Coleta nome, idade e opinião do cliente
- Utiliza estrutura de repetição FOR
- Utiliza estrutura de decisão IF
- Conta quantas respostas foram "EXCELENTE" e "RUIM"

## ▶️ Como executar
1. Abra o terminal
2. Execute:
   python pesquisa_opiniao.py

## 🧪 Testes
Foram realizados testes com 10 entrevistados para validação.
